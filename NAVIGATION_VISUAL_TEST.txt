API ENDPOINTS USED:
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

GET /api/prof/hierarchy
  - Returns hierarchical data structure
  - Called once on component mount via useEffect
  - Response: Filiere[] with nested modules, groupes, seances

POST /api/prof/seances/[id]/open
  - Opens a seance for attendance marking
  - Updates seance status to "OPEN"
  - Allows QR display

POST /api/prof/seances/[id]/close
  - Closes seance for attendance marking
  - Updates seance status to "CLOSED"
  - Enables PDF download

GET /api/prof/seances/[id]/attendance
  - Fetches present/absent students
  - Returns { present: Student[], absent: Student[] }

POST /api/prof/seances/[id]/freeze
  - Freezes/unfreezes QR code
  - Body: { frozen: boolean }
  - Updates qrFrozen state + shows alert

GET /api/prof/seances/[id]/export-pdf
  - Opens HTML report in new tab
  - Browser handles PDF conversion via print dialog
